# Windows Defender Antivirus

## Objective
By completing this lab, you will develop a comprehensive understanding of how to secure a Windows operating system using the functionality of real-time protection provided by Windows Defender.



### Skills Learned
- Review Windows Security Virus and threat protection.
- Update threat definitions.
- Run Windows Defender Antivirus quick scan.
- Run Windows Defender Antivirus custom scan.

### Tools Used
- Windows Defender (Virus & threat protection)
